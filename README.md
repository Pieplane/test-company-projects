## Установка и запуск проекта

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/Pieplane/test-company-projects.git
   ```

2. Установите зависимости:

   ```bash
   npm install
   ```

3. Запустите проект:

   ```bash
   npm start
   ```

4. Перейдите на `http://localhost:3000` для просмотра.
